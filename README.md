##### 대학생들을 위한 자취 정보 제공 서비스 

유니빌(UniVill)은 혼자 사는 대학생들의 보다 편리하고 경제적인 자취 생활을 위해 대학생 1인 가구들 간 정보 공유의 장을 제공합니다.
